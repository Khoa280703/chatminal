## Runtime Architecture
1. `main()` sets process behavior (broken-pipe signal ignored), initializes logging, starts Iced.
2. `AppState::boot()` loads config, resolves runtime font metrics, creates `SessionManager`.
3. `SessionManager::create_session()` opens PTY, spawns child shell, starts reader/writer threads.
4. Reader thread parses PTY bytes with `vte::Parser` into `TerminalGrid`.
5. Reader publishes `SessionEvent::Update` (non-blocking `try_send`, retry on full) and `SessionEvent::Exited` (blocking send on EOF/error).
6. `AppState::subscription()` maps session events and UI events into `Message`.
7. `AppState::update()` mutates UI/session state and forwards keyboard bytes into PTY input queue.
8. `terminal_pane_view` draws terminal cells via Iced `Canvas` with generation-based cache invalidation.

## Core Modules

### `src/app.rs`
- Owns active session, per-session grid snapshots, per-session scroll offsets, runtime cols/rows, and font/cell metrics.
- Applies viewport clamp logic for scroll offset updates.
- Calculates cols/rows from window size and runtime cell metrics, then resizes all sessions.
- Uses `terminal_generation` counter to invalidate canvas cache predictably.

### `src/config.rs`
- Normalizes config values before runtime use.
- Numeric bounds:
  - `scrollback_lines`: `100..=200_000`
  - `font_size`: `8.0..=48.0`
  - `sidebar_width`: `160.0..=640.0`
- Non-finite float values (`f32::NAN`, `f32::INFINITY`) fall back to defaults then clamp.

### `src/session/manager.rs`
- Stores sessions in `IndexMap<SessionId, Session>` to preserve stable ordering in sidebar.
- Enforces PTY input payload cap (`MAX_INPUT_BYTES = 65_536`).
- Maps input queue states to explicit errors: `SessionError::ChannelFull`, `SessionError::ChannelClosed`.
- Clamps PTY resize dimensions to valid `u16` range.

### `src/session/pty_worker.rs`
- Reader thread:
  - Parses PTY bytes into terminal mutations.
  - Sends update snapshots with `try_send`.
  - On `TrySendError::Full`, logs warning and keeps dirty state for retry.
  - On EOF/error, sends `SessionEvent::Exited` with `blocking_send`.
- Parser coverage includes cursor movement, erase ops, style/color SGR, alternate screen, and `ESC M` reverse index.

### `src/session/grid.rs`
- Terminal model: primary + alternate buffers, cursor, scrollback ring buffer.
- Scrollback capacity is clamped and enforced.
- `scroll_down` inserts blank line at top (used by reverse index behavior).

### `src/ui/theme.rs` and `src/ui/terminal_pane.rs`
- `metrics_for_font(font_size)` computes runtime cell metrics:
  - `cell_width = max(font_size * 0.6, 5.0)`
  - `cell_height = max(font_size * 1.35, font_size + 2.0)`
- Terminal canvas maps wheel input into line deltas and renders visible rows from active grid or scrollback.
