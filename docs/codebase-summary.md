## Runtime Flow (Observed)
1. `AppState::boot()` loads config, computes cell metrics (`metrics_for_font`), creates `SessionManager`, opens first session.
2. `SessionManager::create_session()` opens PTY, spawns shell process, starts reader/writer threads.
3. Reader thread calls `Terminal::advance_bytes` in `PtyEngine` and builds `TerminalGrid` snapshots.
4. Update snapshots are sent as `SessionEvent::Update` via `try_send`.
5. If queue is full, worker keeps `dirty = true` and retries latest snapshot on next flush.
6. On EOF/error, worker flushes once and dispatches `SessionEvent::Exited` from a spawned thread via `blocking_send`.
7. `AppState::update()` merges snapshots and uses `lines_added` to preserve user scroll offset.
8. `terminal_pane_view` draws cells/cursor from immutable grid snapshots and generation-based cache invalidation.

## UI Rendering Notes
- Cursor styles rendered: `CursorStyle::Block`, `CursorStyle::Underline`, `CursorStyle::Bar`, `CursorStyle::Hidden`.
- Underline rendering follows cell attributes even when cell text is empty/continuation.
- Default metrics at `font_size = 14.0` from `metrics_for_font` are approximately:
  - width: `8.68`
  - height: `16.8`
- Color mapping:
  - non-default fg/bg colors are resolved via wezterm palette (`resolve_fg` / `resolve_bg`)
  - renderer maps `CellColor::Rgb` directly to Iced colors.
